# Versus UI Library — Public API Surface (locked, reference only)
Authoritative docs: https://versusairlines.top/developers.html

**Load:** `local Library = loadstring(game:HttpGet("https://versusairlines.top/scripts/NewLibrary.lua"))()`

**Window:** `local ui = Library:Setup({ Location = game.CoreGui, OpenCloseLocation = "Top Center" })` — supported positions: Top/Middle/Bottom × Left/Center/Right.

**Sections:** `ui:CreateSection("Versus")` returns a section object that supports:
- `createButton({ Name, Description, Callback, VersusAI = true, flagName })` — VersusAI adds the AI Pathing consent card.
- `createToggle({ Name, Description, Flag, flagName, Callback, Warning = function() ... end, WarnIf = function() ... end })`
- `createSlider({ Name, Description, flagName, value, minValue, maxValue, Callback })`
- `createInputBox({ Name, flagName, Flag, Callback })`
- `createDropdown({ Name, flagName, Flag, List, multi, Callback })` → returns `dd` with `dd:updateList(...)`.
- `createLabel({ Name, Special, TransparentBackground, Center, Bold, Underline, Italic, Markup })` → label can `:Set(...)`.

**Library-level:**
- `Library.Flags` — flat table of all saved flags; auto-persists to `saves.json` per placeId when executor supports FS.
- `Library.isClosed` — read-only boolean; window visibility.
- `Library.DisableSounds`, `Library:PlaySound("Click"|"Pop")`.
- `Library:TrackConnection(rbxScriptConnection, tag)`, `Library:CleanupConnectionsByTag(tag)`, `Library:CleanupConnectionsByPrefix(prefix)`, `Library:CleanupConnections()`, `Library:Cleanup()`.
- `Library:createDisplayMessage(title, desc, buttons, style)` — style = `info|warning|danger`.
- `Library:PromptBugReport()`.
- `Library:UpdateUI("Dark Mode")` (preset), `Library:OpenCustomizer()`, `Library:SaveCurrentThemeAs(name)`, `Library:OverwriteSavedTheme(name)`.
- `Library:GetDeclaredConfigDefaults()`, `Library:ClearResidualBlur()`, `Library:CloseAllPopups()`, `Library:GetActiveModalCount()`.
- Theme presets: Light, Dark, Halloween, +20 others; user themes in `Versus Airlines 2.0/my_themes.json`; presets in `Versus Airlines 2.0/themes.json`.

**VersusAI pathing:** opt-in via `createButton({ VersusAI = true, ... })`. Use `Library.Flags["UseVersusAI"]` to branch between AI-learned routes and the script's normal movement. No `PathfindingService` involved.

**Filesystem:** `Library` itself writes `Versus Airlines 2.0/saves.json` per placeId when FS is available. Folder location is fixed.
